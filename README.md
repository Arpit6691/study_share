📚 StudyShare – Smart Study Material Sharing Platform

StudyShare is a full-stack web application that allows students to upload, manage, preview, and share study materials seamlessly. It is designed to provide a smooth and efficient way to organize academic resources with a modern UI and real-world functionality.

🚀 Live Demo

👉 https://study-share-dun.vercel.app/

✨ Features
🔐 Authentication
Secure user login/signup
Google OAuth integration
📂 File Management
Upload study materials (PDFs, images, etc.)
View and manage your uploaded files
Delete files anytime
👀 File Preview
In-app preview for documents and images
No need to download files to view content
🔎 Search & Filters
Search files by name
Filter files for better organization
👤 User Profile
Dedicated profile page
View your activity and uploads
📊 Activity Tracking
Track uploaded files
Monitor user activity/history
🎨 Modern UI/UX
Clean and responsive design
Dark theme interface
Smooth navigation
